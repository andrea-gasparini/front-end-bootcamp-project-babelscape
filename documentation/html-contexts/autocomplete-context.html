Documentazione Autocomplete

L’Autocomplete è un componente che permette di mostrare dei suggerimenti per il completamento automatico di una stringa.

È formato da due elementi principali:

  • L'input box, in cui l'utente può inserire una stringa di testo per cui saranno mostrati i suggerimenti
  • Il body, ovvero il menù a tendina che mostra i suggerimenti disponibili per la stringa inserita

Quando l'utente clicca su uno dei suggerimenti disponibili presenti nel body, questo verrà inserito all'interno dell'input box. In caso la stringa in input non abbia alcun suggerimento, il body non sarà mostrato.

Esempi

Il componente può essere personalizzato in larghezza, nel placeholder, nel numero minimo di caratteri per mostrare i suggerimenti e nell'URL del servizio web che fornisce i suggerimenti.

A seguire alcuni esempi con la relativa configurazione. 

$("#esempio1").autocomplete({
    width: 200,
    restURL: new URL('http://localhost:30000/autocomplete?word=')
});

$("#esempio2").autocomplete({
    placeholder: 'Autocompletamento..',
    minTextLength: 5,
    restURL: new URL('http://localhost:30000/autocomplete?word=')
});

Opzioni

Le opzioni del componente vanno specificate nella configurazione tramite un oggetto JSON, come mostrato negli esempi precedenti. Se l'opzione non ha alcun valore in default è obbligatorio specificarne uno per la corretta configurazione del componente.

Opzione Tipo Default Descrizione
placeholder string 'Inserisci del testo..' La stringa che viene mostrata nell'input box del componente quando non viene scritto nulla.
width number auto Larghezza del componente, di default si adatta alle dimensioni del div in cui è contenuto.
restURL URL L'indirizzo del servizio web che, data una stringa come parametro in input, restituisce la lista delle parole più simili.
minTextLength number 1 Il minimo numero di caratteri per permettere al componente di effettuare una richiesta al servizio web.
