Documentazione Toggle

Il Toggle è un componente che permette di fare lo switch tra due valori.

Contiene il bottone principale, che se cliccato cambia la selezione, e due label relative ai valori selezionabili. Tipico esempio è il bottoncino ON/OFF.

Esempi

Oltre che nei valori delle label, il componente può essere personalizzato nelle dimensioni e nella funzione di callback che viene invocata ogni volta che la selezione cambia.

A seguire alcuni esempi con la relativa configurazione. 

$("#esempio1").toggle({
    value1: { id: 'on', label: 'ON'},
    value2: { id: 'off', label: 'OFF' },
    dataMapper: (v) => { key: v.id, value: v.label },
    onChange: (v) => console.log("Toggle value: " + v.value)});

$("#esempio2").toggle({
    value1: 'SI,
    value2: 'NO',
    color1: 'blue',
    color2: 'red',
    dataMapper: (v) => { key: v, value: v }});

$("#esempio3").toggle({
    value1: { id: 'accept', label: 'ACCEPT'},
    value2: { id: 'decline', label: 'DECLINE' },
    color1: 'lightblue',
    height: 40,
    dataMapper: (v) => { key: v.id, value: v.label }});

Opzioni

Le opzioni del componente vanno specificate nella configurazione tramite un oggetto JSON, come mostrato negli esempi precedenti. Se l'opzione non ha alcun valore in default è obbligatorio specificarne uno per la corretta configurazione del componente.

Opzione Tipo Default Descrizione
value1 T Il primo valore del toggle, che verrà mostrato nella label di sinistra.
value2 T Il secondo valore del toggle, che verrà mostrato nella label di destra.
height number 25 L'altezza del componente. Al variare di questa vengono modificate di conseguenza anche le dimensioni dei label laterali.
color1 string lightgreen Colore del componente se è selezionato il primo valore. Può essere rappresentato come stringa testuale, in formato RGB o esadecimale. Es: #90EE90, rgb(144, 238, 144).
color2 string grey Colore del componente se è selezionato il secondo valore. Può essere rappresentato come stringa testuale, in formato RGB o esadecimale. Es: #808080, rgb(128, 128, 128).
dataMapper function<T, KeyValue> Una funzione per mappare value1 e value2 in un oggetto KeyValue.
onChange function<KeyValue> (v) => console.log( "Valore selezionato: " + v.value ) Funzione (callback) che viene invocata ogni volta che la selezione del toggle cambia.

Metodi

Il componente dispone inoltre dei metodi descritti a seguire.

.toggle('setSecondValue')

Imposta il toggle sul secondo valore (value2).

$("#esempio2").toggle('setSecondValue');
→

.toggle('setFirstValue')

Imposta il toggle sul primo valore (value1).

$("#esempio2").toggle('setFirstValue');
→

.toggle('getValue')

Ritorna il valore selezionato del toggle.

let selectedValue = $("#esempio2").toggle('getValue');
