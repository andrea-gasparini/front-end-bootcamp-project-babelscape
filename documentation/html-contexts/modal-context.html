Documentazione Modal

La Modal è un componente che permette di mostrare una finestra in rilievo.

Si può utilizzare per evidenziare particolari condizioni in cui si trova la UI in quel momento. Il componente viene mostrato sopra l'intera interfaccia, disabilitando la possibilità di cliccare sugli elementi della pagina finché non viene chiuso. È disponibile in due diverse tipologie:

  • DIALOG, in cui viene mostrato un messaggio
  • CONFIRMATION, in cui viene chiesto all'utente di confermare oppure annullare un operazione

Il componente è formato da tre elementi principali:

  • L'header, che rappresenta il titolo della Modal
  • Il body, che rappresenta il corpo centrale della Modal
  • Il footer, in cui sono presenti i bottoni per interagire con la Modal

Esempi

Oltre che nella tipologia, il componente può ovviamente essere personalizzato nel contenuto dell'header e del body. Inoltre può essere definita la funzione che viene invocata nel caso in cui la Modal sia di tipo CONFIRM e l'utente abbia confermato l'operazione.

A seguire alcuni bottoni che mostrano delle Modal di esempio, con la relativa configurazione sul click del bottone. 

$('<div />').appendTo('body').modal({
    header: "Header title della prima Modal",
    body: "Lorem ipsum dolor sit amet, consectetur adipiscing elit [...]"
});

$('<div />').appendTo('body').modal({
    type: "CONFIRM",
    body: $('<div />')
        .append(
            'Questa Modal non ha un Header ed è della tipologia',
            ' <code>CONFIRM</code>. ',
            'Cliccando sul bottone di conferma verrà avviata la funzione onConfirm.'
            ),
    onConfirm: () => console.log('Hai cliccato su conferma.')
});

Opzioni

Le opzioni del componente vanno specificate nella configurazione tramite un oggetto JSON, come mostrato negli esempi precedenti. Se l'opzione non ha alcun valore in default è obbligatorio specificarne uno per la corretta configurazione del componente.

Opzione Tipo Default Descrizione
type ModalType DIALOG La tipologia della modal (è possibile scegliere tra DIALOG e CONFIRM).
header string o JQuery '' L'header della modal, può essere specificato tramite stringa di testo o codice JQuery.
body string o JQuery '' Il body della modal, può essere specificato tramite stringa di testo o codice JQuery.
onConfirm function () => console.log('conferma!') Funzione (callback) che viene invocata solamente nel caso in cui la modal sia di tipo CONFIRM e l'utente confermi l'operazione.

Metodi

Il componente dispone inoltre dei metodi descritti a seguire.

.modal('dispose')

Rimuove la modal dal DOM.

$("#modal-id").modal('dispose');
