Documentazione Table

Il Table è un componente molto semplice che visualizza una serie di dati presi in input in fase di creazione.

A differenza delle semplici tabelle questo componente dispone delle seguenti funzionalità:

  • Ordinare le righe in ordine alfabetico in base ai valori delle singole colonne
  • Conteggio del numero di righe totali
  • Somma totale dei valori delle colonne numeriche

Esempi

Il componente Message è disponibile in 4 principali tipologie di messaggio: INFO, SUCCESS, WARNING e DANGER (la tipologia INFO è di default se non ne viene specificata una diversa). Il componente può essere personalizzato anche nelle dimensioni e nell'allineamento del testo al proprio interno.

A seguire gli esempi di base con la relativa configurazione. 

$('#esempio-info').message({
message: 'Un semplice Message Info di test'});

$('#esempio-success').message({
message: 'Un semplice Message Success di test',
type: 'SUCCESS'});

$('#esempio-warning').message({
message: 'Un semplice Message Warning di test',
type: 'WARNING'});

$('#esempio-danger').message({
message: 'Un semplice Message Danger di test',
type: 'DANGER'});

Opzioni

Le opzioni del componente vanno specificate nella configurazione tramite un oggetto JSON, come mostrato negli esempi precedenti. Se l'opzione non ha alcun valore in default è obbligatorio specificarne uno per la corretta configurazione del componente.

Opzione Tipo Default Descrizione
message string Messaggio di testo da mostrare nel componente.
width number auto Larghezza del componente, di default si adatta alle dimensioni del div in cui è contenuto.
height number auto Altezza del componente, di default si adatta alle righe del messaggio.
type Alert o string INFO Tipologia del messaggio, che modifica i colori del componente. È possibile scegliere tra INFO, SUCCESS, WARNING e DANGER
alignText Align o string LEFT Allineamento del messaggio all'interno del componente. È possibile scegliere tra LEFT, CENTER e RIGHT
