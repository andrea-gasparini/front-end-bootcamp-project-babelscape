Documentazione Table

Il Table è un componente che visualizza una serie di dati presi in input in fase di creazione.

A differenza delle semplici tabelle questo componente dispone delle seguenti funzionalità:

  • Ordinare le righe in ordine alfabetico in base ai valori delle singole colonne
  • Conteggio del numero di righe totali
  • Somma totale dei valori nelle colonne numeriche

Esempi

Il componente può essere personalizzato nell'utilizzo o meno della prima riga come header (questa opzione viene utilizzata di default) nelle dimensioni e nell'allineamento del testo al proprio interno (impostato a CENTER se non ne viene specificato uno diverso).

A seguire alcuni esempi con la relativa configurazione. 

$("#esempio1").table({
    alignText: 'left',
    data: [
        ['Nome', 'Cognome', 'Sesso', 'Figli a carico'],
        ['Mario', 'Rossi', 'M', 2],
        ['Roberto', 'Bianchi', 'M', 0],
        ['Giovanni' ,'Verdi' , 'M', 4],
        ['Maria' ,'Neri' , 'F', 2]]});

$("#esempio-no-header").table({
    firstRowHeader: false,
    data: [
        ['mele', '0,89 € / Kg', 'Melinda', 5000, 'Roma'],
        ['pere', '1,05 € / Kg', 'Perelle', 2600, 'Roma'],
        ['arance', '0,67 € / Kg', 'SicilyDoc', 12050, 'Milano'],
        ['pesche' , '0,34 € / Kg', 'Peschine', 520, 'Napoli'],
        ['kiwi', '0,99 € / Kg', 'TopKiwi', 430, 'Roma']]});

Opzioni

Le opzioni del componente vanno specificate nella configurazione tramite un oggetto JSON, come mostrato negli esempi precedenti. Se l'opzione non ha alcun valore in default è obbligatorio specificarne uno per la corretta configurazione del componente.

Opzione Tipo Default Descrizione
data Array<Array<T>> La matrice dei dati da mostrare nel componente.
width number auto Larghezza del componente, di default si adatta alle dimensioni del div in cui è contenuto.
height number auto Altezza del componente, di default si adatta alle righe della tabella.
firstRowHeader boolean true Definisce se la prima riga deve essere mostrata come header (se true) o meno (se false). La presenza dell'header determina la possibilità di ordinare le righe in base ai valori delle singole colonne.
alignText Align o string LEFT Allineamento del testo all'interno del componente. È possibile scegliere tra LEFT, CENTER e RIGHT
