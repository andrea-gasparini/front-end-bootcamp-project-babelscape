Documentazione Message

Il Message Ã¨ un componente molto semplice, che si occupa di mostrare un messaggio preso in input.

Esempi

I componenti Message sono disponibili in 4 principali tipologie di messaggio: INFO, SUCCESS, WARNING e DANGER. Possono essere personalizzati anche nelle dimensioni e nell'allineamento del testo al loro interno. 

$('#esempio-info').message({
        message: 'Un semplice Message Info di test'});

$('#esempio-success').message({
        message: 'Un semplice Message Success di test',
        type: 'SUCCESS'});
    
$('#esempio-warning').message({
        message: 'Un semplice Message Warning di test',
        type: 'WARNING'});

$('#esempio-danger').message({
        message: 'Un semplice Message Danger di test',
        type: 'DANGER'});
